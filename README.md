# Descripción del proyecto

"Hasta la Última Parada" es un prototipo de videojuego en 2D inspirado en una experiencia de supervivencia dentro de un sistema de transporte público.  
El jugador debe reaccionar a diferentes amenazas y sobrevivir a múltiples ataques enemigos mientras avanza entre niveles.

Esta primera versión implementa las mecánicas principales del juego, incluyendo movimiento del jugador, sistema de ataques, enemigos funcionales y una interfaz básica.
